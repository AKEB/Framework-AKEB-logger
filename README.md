## Install

composer project akeb/logger

Composer config

```json
{
    "require": {
        "akeb/logger": "^1.0.0"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://git.terrhq.ru/v.babajanyan/Logger"
        }
    ]
}
```

or

```json
{
    "require": {
        "akeb/logger": "^1.0.0"
    },

            "type": "vcs",
```
