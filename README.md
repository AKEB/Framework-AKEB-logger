# Logger

## Install

composer project akeb/logger

Composer config

```json
{
    "require": {
        "akeb/logger": "^1.2.0"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/AKEB/Logger"
        }
    ]
}
```

or

```bash
composer require akeb/logger
```
